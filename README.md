# Hotel equipment management

Hotel equipment web app is created to help hotel manage the distribution of its equipments, for example: Laptop, mouse, keyboard, printer, etc..
\**The advantage of this project are :**\
1.Reusability of the working equipment that were replaced and sent to the storage.
2.Keep track of the equipment during while it's in use, this act would minimize the chances of stealing. Since the app __saves the name of the user of each device in every department__.

**Technology used** : HTML, CSS, BOOTSTRAP, PHP, MYSQL, MVC design pattern.\
